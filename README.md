# GetIPFromQQ
**通过获取UDP中QQ电话数据包特征，捕获对方IP地址**

**注意：对方可不接通电话，但对方必须响铃才可获取，需要目标对方在线。**

## 使用指南

1. 运行软件，选择监听的网卡
2. 拨通目标QQ电话
3. 等待程序输出目标IP
